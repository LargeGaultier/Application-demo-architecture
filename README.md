# Application de Démonstration d'Architectures

Application de test pour démontrer, avec le moins de packages possible, différentes approches d'architecture :

- **N-layer**
- **Clean Architecture**

Il y a également un exemple d'application de microservice très simple pour illustrer des cas simplifiés de transactions distribuées.

## Comment utiliser ce dépôt

Chaque exemple d'architecture est disponible sur une branche distincte. Il vous suffit de cloner le dépôt et de basculer sur la branche correspondante.
